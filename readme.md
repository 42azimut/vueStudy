### 03.핵심키워드 v-bind
`<img v-bind:src="imgeSource" alt="random" />`  #imageSource를 v-bind 디렉티브(directive) 즉, 데이터를 바인딩(엮는다)을 사용하여 변수처럼 사용 가능!
